What is this

This is an interactive-story building tool, like "Twine".

How to make a link

To link passages together, use the following syntax:
[[display text|passage title]] or [[passage title]]

For example, writing "Make your choice! [[red]] [[blue]] [[green]]" will link to three different passages titled "red", "blue" and "green".

However if you want to use a different text, for example "Push the DOOM button? [[push|bad ending]] [[don't|bad ending]]" you can display link text that does not match title of the passage that will be linked.

How to play my game?

You can test your game by unchecking the "Edit" toggle, or you can experience it as a player by pressing the "Share" button.

How to save my work

The download button allows you to download a file of your game data. You can restore it with the upload button.
